CPU Pipeline Simulator

The purpose of this program is too further understand the inner workings of a CPU in one of the simplest forms.

Structure: Created multiple classes to keep track of the necessary information for the given instruction at the current stage. Such as IF, ID, EXE, MEM, WB, etc., each one will store the essential information for that instruction. Created class Stage and made two instances of the class. One instance holds the current cycle information; The second instance contains the following cycle information.

Outcome: With the completion of this project helped me further understand the inner workings of a CPU. Additionally, have tremendous respect for those working with computer architecture.
